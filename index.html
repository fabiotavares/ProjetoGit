Projeto Git - Fase 1

Conforme solicitado, abaixo seguem os comandos utilizados para realizar um push em meu repositório:

$ git remote add origin git@github.com:fabiotavares/ProjetoGit.git
$ git clone https://github.com/fabiotavares/ProjetoGit.git
$ git status
$ git add arquivo1.php
$ git add arquivo2.php
$ git add arquivo3.php
$ git commit -m "Adicionando arquivos 1, 2 e 3"
$ git log
$ git push origin master
